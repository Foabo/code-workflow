# TypeScript and Node.js for version 1

CW version 1 will be implemented in TypeScript on Node.js. The product is primarily a repository-local file workflow tool that reads and writes JSON, JSONL, Markdown, templates, and coding-harness adapter files, so TypeScript provides a good balance of development speed, type safety, package distribution, and ecosystem fit. Lower-level languages such as Go or Rust remain possible later if packaging or performance requirements justify a rewrite or native helper.
