# Agent commands use the cw prefix

CW-generated agent commands will use a `cw-` prefix, such as `cw-work`, `cw-clarify`, `cw-plan`, `cw-run`, `cw-check`, `cw-finish`, `cw-resume`, `cw-discard`, `cw-doctor`, and `cw-understand`. The prefix prevents collisions with commands from coding harnesses, plugins, and other workflow frameworks.
