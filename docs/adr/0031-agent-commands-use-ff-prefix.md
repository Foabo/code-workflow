# Agent commands use the ff prefix

Flowflow-generated agent commands will use a `ff-` prefix, such as `ff-work`, `ff-clarify`, `ff-plan`, `ff-run`, `ff-check`, `ff-finish`, `ff-resume`, `ff-discard`, `ff-doctor`, and `ff-understand`. The prefix prevents collisions with commands from coding harnesses, plugins, and other workflow frameworks.
